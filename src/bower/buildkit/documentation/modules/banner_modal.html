Banner Modal

The banner modal displays in the footer of the page. It is dismissable by any button with the CSS class name
js-banner-modal-btn-close.

To only show the modal to certain locales, add a data-options attribute:


      data-options="showToLocales:de,es,fr,en-gb,it,nl"

Valid locales are:

  • Australia: en-au
  • Brazil: pt-br
  • Canada (English): en-ca
  • Canada (French): fr-ca
  • France: fr
  • Germany: de
  • Italy: it
  • Netherlands: nl
  • Spain: es
  • US: en
  • UK: en-gb
