Promo Banner

A vertically responsive banner to be used anywhere on the page to post current marketing campaigns.

You will need to provide 2 assets for the cinema display, promo-banner__img--cinema and mobile promo-banner__img--mobile to set the image paths.

Default settings

Device Breakpoints Suggested image dimensions
Mobile < 750px (default) 768px × 329px
Tablet / Desktop* 906px > 623px × 169px
