Hero Banner

A horizontally and vertically responsive banner intended to appear at the top of pages. We use separate images for mobile and tablet/desktop* displays.

To set up images for mobile or tablet/desktop*, add data-sm-src and data-lg-src attributes to hero-banner__container and set these to image paths. data-sm-src correlates to mobile; data-lg-src to tablet/desktop.

Keep in mind that asset dimension recommendations accommodate retina displays. Javascript code on the client-side will size the images automatically depending on the display size.

Images should follow the recommended dimensions or maintain the same aspect ratio as the recommended dimensions.

* We use the same image for both tablet and desktop

Default settings

Device Breakpoints Suggested image dimensions
Mobile < 750px (default) 800px × 1050px
Tablet / Desktop* 750px > 1960px × 680px

Cinema Display

The tablet and desktop versions display the hero image at the full browser width. Images can have any dimension, but consider following the suggested dimensions below. Use the CSS class hero-banner--cinema.

Device Breakpoints Suggested image dimensions
Mobile < 750px (default) 800px × 1050px
Tablet / Desktop* 750px > 2136px × 680px

* We use the same image for both tablet and desktop
