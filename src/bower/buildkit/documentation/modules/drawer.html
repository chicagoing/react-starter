Drawer

The top drawer with images, text and cta buttons. You can select from 1, 2, 3, 4 or 5. By default, the heading will use all-caps, but add typ__letter-case to use letter case (see Custom alignment below for an example). All columns are left aligned by default. You have the options to add buttons, and have the bottom drawer open or closed.

You can link modules within the drawer module by adding data-options="enableTabLinking:true" as a param to pass into the drawer partial. You will need to add ids to the drawer, ie. id="tab1" and references to the tabs in the drawer_options, ie. href="#tab5" and a new class js-tab-changer on the same link. See drawer with 5 options for an example.

Note: This module uses styles from the Grid module.
