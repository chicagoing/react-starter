Multi Column CTAS

Multi columns with icons, text and cta button. You can select from 1, 2, 3, 4 or 5. All columns have text and buttons left aligned by default.

Size of assets for the images is limited by width but can by any height.

Different modifications available for the grid module:

  1. Number of grids - add class name (grid--five, grid--four, grid--three, grid--two) to the outer most div
  2. Background color - For a gray background, add the v-bg-white-smoke class to the outer most div container.
  3. Alignment - To right align a column, add the grid__column--right class to the same section as the "grid__column". To center align a column, add the grid__column--center class to the same section as the "grid__column".
  4. Typography - By default, the heading will use all-caps, but add typ__letter-case to the same element as grid__header to use letter case. See Five Column below for an example.
  5. Different spacing for icons - Add grid__icons to the outermost div. See Three Column Icons below for an example.

Maintaining a single column for mobile

Apply the class .grid--mobile-single-col if you want to display only one column for mobile. This is useful when you're embedding other modules inside the grid that might not work so well on mobile.
